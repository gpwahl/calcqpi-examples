How to Cite this Project
========================

If you publish work using calcQPI and/or building on the minimal models, please cite the paper:

- P. Wahl, L.C. Rhodes, C.A. Marques, *calcQPI: : A versatile tool to simulate quasi-particle interference*, [SciPost Physics Codebases 61 (2025)](https://dx.doi.org/10.21468/SciPostPhysCodeb.61).
